# simple-text-Area

Live Demo : https://shyxmmmmmm.github.io/simple-text-Area/


📌 Project: Simple Textarea Message Box

🧑‍💻 This project is a simple Textarea Message Box built using HTML, Tailwind CSS, and JavaScript.

✨ Features:

• 📝 Textarea input for typing messages

• 🔢 Live character counter (0 / 200)

• ⚠️ Maximum 200 characters limit

• 📋 Copy button to copy the typed message

• 🧹 Clear button to erase the message instantly

• ⏱ Success message shown when text is copied

• 📱 Responsive design using Tailwind CSS

🛠 Technologies Used:

• HTML5 – structure of the message box

• Tailwind CSS – styling and responsive layout

• JavaScript – character counting and button actions

• Font Awesome – icons for Copy and Clear buttons

⚙️ How it works:

1️⃣ User types a message inside the textarea

2️⃣ Character counter updates in real time

3️⃣ Text is limited to a maximum of 200 characters

4️⃣ Copy button shows a "Message Copied!" notification

5️⃣ Clear button removes all text and resets the counter

🎯 Purpose of the Project:

This project helps practice JavaScript DOM manipulation, real-time input handling, character counting, and creating interactive UI components using Tailwind CSS.
